# KUCBT Score Analysis

An Exploratory Data Analysis (EDA) project on the Kathmandu University Common Admission Test (KUCBT) scores. This project analyzes publicly available admission data from Kathmandu University's published score sheets to uncover trends in admissions across different engineering departments.

## Features

- Parsed KUCBT score sheets directly from PDFs using **Tabula**.
- Cleaned and standardized raw admission data for analysis.
- Analyzed admission data for **2022**, **2023**, and **2024** across all engineering departments.
- Included **2025** analysis for **Computer Engineering** and **Civil Engineering**.
- Compared admission cutoff scores across departments and years.
- Analyzed gender distribution for each department.
- Studied the competitiveness of the **Computer Engineering** program through score distributions and cutoff trends.
- Generated visualizations and summary statistics to highlight admission patterns.

## Technologies Used

- Python
- Pandas
- NumPy
- Tabula
- Matplotlib

## Dataset

The dataset was obtained from **publicly available KUCBT score sheets** published on the Kathmandu University website. Since the data was distributed as PDF documents, the project involved:

- Extracting tabular data from PDFs.
- Cleaning and standardizing the extracted data.
- Preparing the dataset for exploratory data analysis.

## Analysis Performed

- Department-wise cutoff score analysis.
- Year-wise comparison of admission trends.
- Gender ratio analysis across departments.
- Competitiveness analysis of the **Computer Engineering** program.
- Summary statistics and visualizations of admission data.
